Kafka Toolkit > com.ibm.streamsx.kafka 3.0.0

General Information

The Kafka toolkit integrates IBM Streams with Apache Kafka brokers. It can be used with following broker versions:

  • 0.10.2 (with restrictions: isolation.level=read_uncommitted must be set for the KafkaConsumer, and the guaranteeOrdering parameter of the KafkaProducer cannot be used)
  • 0.11
  • 1.0
  • 1.1
  • 2.0
  • 2.1
  • 2.2
  • 2.3

All Kafka versions below 2.3 do not support the staticGroupMember parameter of the KafkaConsumer operator.

Additional information

Additional user documentation can be found in the toolkit documentation on GitHub.
